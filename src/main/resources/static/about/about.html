About AEDES

A smart, open-source facility reservation system designed for universities to efficiently manage campus resources.

Features

Dynamic Campus Map

Admin creates campuses, buildings, and rooms. System auto-generates the UI — no hardcoded layouts.

Smart Auto-Assignment

Intelligent algorithm filters eligible rooms and automatically assigns the best match based on scoring.

Visual Schedule

View room availability at a glance with intuitive calendar interface and weekly schedules.

Role-Based Access

Tailored interfaces for students, faculty, and administrators with appropriate permissions.

Analytics Dashboard

Room utilization heatmaps, peak hour detection, and per-building statistics.

Secure Authentication

Industry-standard JWT authentication with role-based authorization ensuring data security.

Technology Stack

Frontend: Vanilla HTML, CSS, JavaScript
Backend: Spring Boot 3.x (Java 17+)
Database: MySQL 8.x
Auth: JWT with Spring Security
